# SD Output 엑셀 서식 명세서

> 이 문서는 `excel_handler.py`가 생성하는 SD Output 엑셀 파일의 **정확한 서식 기준**을 정의합니다.
> 서식이 깨졌거나 재구현이 필요할 때 이 문서를 기준으로 복구합니다.
>
> **기준 샘플 파일:** `spams/MMSC스팸추출_20260414_A.xlsx`

---

## 1. 공통 규칙

### 1-1. 폰트
| 적용 대상 | 폰트명 | 크기 | 굵기 |
|-----------|--------|------|------|
| 헤더 행 (1행) | 맑은 고딕 | 10pt | **Bold** |
| 데이터 행 | 맑은 고딕 | 10.5pt | 보통 |

### 1-2. 헤더 행 (1행) 공통 서식
- 가로 정렬: **가운데**(`center`)
- 세로 정렬: **가운데**(`center`)
- 배경색: `#D3D3D3` (연회색)

### 1-3. 데이터 행 기본 서식
- 가로 정렬: **설정하지 않음** — 엑셀 기본값 적용 (숫자 자동 우측, 문자 자동 좌측)
- 세로 정렬: **가운데**(`center`)

### 1-4. 분류 컬럼 (공통)
- 저장 타입: **정수(`int`)** — 문자열로 저장하면 엑셀이 좌측 정렬로 표시함
- 가로 정렬: 별도 설정 없음 (int 타입이면 자동으로 우측 정렬)
- 코드 의미:

  | 코드 | 분류 |
  |------|------|
  | `0` | 기타 스팸 |
  | `1` | 유해성 스팸 |
  | `2` | 사기/투자 스팸 |
  | `3` | 불법 도박/대출 |

### 2-2. 육안분석(시뮬결과35_150)

**담당 함수:** `_sort_sheet_by_type`, `_apply_formatting`

#### 컬럼 구성

| 열 | 헤더 | 컬럼 너비 | 데이터 가로 정렬 |
|----|------|-----------|-----------------|
| A | 메시지 | 90.0 | 없음 (좌측) |
| B | URL | 22.0 | 없음 (좌측) |
| C | 구분 | 10.6 | 가운데 |
| D | 분류 | 6.6 | 없음 (int → 자동 우측) |
| E | 메시지 길이 | 12.6 | 가운데 |
| F | URL 길이 | 10.6 | 가운데 |
| G | Probability | 10.0 | 가운데 |
| H | Semantic Class | 15.0 | 가운데 |
| I | Reason | 90.0 | 없음 (좌측) |
| J | Red Group | 13.0 | 가운데 |

#### 행 배경색

| 조건 | 배경색 | 색상 코드 |
|------|--------|-----------|
| SPAM 행 (구분='o', Type A/B) | 황금색 | `#FFFFF2CC` |
| Red Group 행 또는 텍스트HAM+악성URL 분리 행 | 연분홍 | `#FFFFE0E0` 계열 |
| HAM 행 | 없음 | — |

#### 정렬 순서

3개의 키를 복합 적용하여 **단일 `sort()` 1회**로 처리:

```
(그룹 순위, URL 유무, 메시지 가나다순)
```

| 순위 | 키 | 값 |
|------|----|----|
| 1순위 | 그룹 | 0: SPAM / 1: Red Group·분리 / 2: HAM / 3: 기타 |
| 2순위 | URL 유무 | 0: URL 있음 → 1: URL 없음 |
| 3순위 | 메시지 | 가나다 오름차순 (A→Z) |

> **Red Group 판단 기준:** `Red Group` 컬럼 값이 `'O'`인 경우 (Reason 텍스트 파싱 아님)

| D~S | (빈 열) | 13.0 | — |

## 3. 자주 깨지는 서식 체크리스트

| 확인 항목 | 올바른 상태 | 잘못된 상태 |
|-----------|------------|------------|
| 분류 컬럼이 좌측 정렬 | `int` 타입으로 저장됨 | `str` 타입('2', '3')으로 저장됨 |
| 문자열 컬럼(문자문장차단등록 B열)이 좌측 정렬 | `horizontal='center'` 적용 | 정렬 미적용 |
| 문자열/문장 중복제거 데이터 행 세로 정렬 없음 | `vertical='center'` 적용 | 폰트·정렬 코드 누락 |
| 육안분석 URL 있는 항목이 URL 없는 항목과 섞임 | 복합 키 단일 sort 사용 | 3회 독립 sort 사용 |
